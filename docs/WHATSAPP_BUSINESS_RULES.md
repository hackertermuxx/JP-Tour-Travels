# WhatsApp Business Rules

## Booking Flow
1. User clicks WhatsApp button (fixed/package/CTA)
2. Pre-filled message opens in WhatsApp
3. Admin receives and responds manually
4. Booking is confirmed via WhatsApp conversation

## Message Templates

### Package Inquiry
```
Hi! I'm interested in the *[Package Name]* package.
Please share more details including pricing and availability.
```

### Quick Booking
```
Hi! I want to book a trip to [Destination/General].
Please share available packages and pricing.
```

### Contact Form
```
*New Contact Inquiry*
Name: [Name]
Email: [Email]
Phone: [Phone]
Message: [Message]
```

## Business Rules
- All bookings go through WhatsApp (primary channel)
- No online payment integration — payment handled offline
- Admin manually tracks bookings via WhatsApp and admin dashboard
- Response time target: Within 1 hour during business hours
